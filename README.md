# Retail Sales Analysis & Prediction Project

## 📊 Project Overview

This project analyzes real-world retail sales data to uncover business insights, identify sales trends, understand customer purchasing behavior, and predict future sales using Machine Learning techniques.

The project follows the complete Data Science lifecycle:

* Data Collection
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Feature Engineering
* Machine Learning Model Building
* Model Evaluation
* Business Insights & Recommendations

---

## 🎯 Objectives

* Analyze retail sales performance.
* Identify top-performing products and categories.

## 📋 Dataset Information

The dataset contains retail transaction records.

### Features

| Feature     | Description             |
| ----------- | ----------------------- |
| Order_ID    | Unique order identifier |
| Date        | Purchase date           |
| Customer_ID | Customer identifier     |
| Product     | Product name            |
| Category    | Product category        |
| Quantity    | Number of units sold    |
| Price       | Product price           |
| Revenue     | Total sales amount      |
